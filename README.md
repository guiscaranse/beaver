## Instalando

Depende de:
* [Python 3](https://www.python.org/downloads/) (3+)
* [Pipenv](https://github.com/pypa/pipenv) (11.8+)

Instale as dependências do pipenv e inicie o ambiente virtual:
```sh
pipenv install
pipenv shell
```

É possível executar o arquivo de testes usando:
```sh
python run.py
```
